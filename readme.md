# ECMAScript 7(ES7) Demo
## Intro
* Based on [ES7 compatibility table][1]
* Demo on [babel REPL][2] and [Chromium][3]

## To use
  >> yum install babel
  >
  > or
  >
  >> npm install -g babel
  >
  > then
  >
  >for babel
  >
  >> babel-node --stage 0 <source>
  >
  >for chromium, just using Developer Console Tools is OK.
